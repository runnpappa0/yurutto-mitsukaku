- **シート名**: `相談依頼`
- **ロケール**: 日本語
- **タイムゾーン**: Asia/Tokyo

## フォーマット設定

### 1. ヘッダー行（1行目）
- **背景色**: #2d4259（セカンダリカラー）
- **文字色**: 白
- **フォント**: 太字
- **サイズ**: 10pt
- **配置**: 中央揃え
- **行の高さ**: 40px

### 2. 固定設定
- **固定行**: 1行目（ヘッダー）
- **固定列**: A列とB列（受信日時・ステータス）

設定方法:
1. A1セルを選択
2. 「表示」→「固定」→「2行目まで」
3. B1セルを選択
4. 「表示」→「固定」→「C列まで」

### 3. B列（ステータス）- データの入力規則
1. B2:B1000を選択
2. 「データ」→「データの入力規則」
3. 条件: 「リストを範囲で指定」
4. リスト項目:
   - 未対応
   - 対応中
   - 完了
5. 「無効なデータの場合」: 入力を拒否
6. 「セルにプルダウンリストを表示」: チェック

### 4. M〜Q列（金額）- 表示形式
1. M2:Q1000を選択
2. 「表示形式」→「数値」→「通貨」
3. 形式: `¥#,##0`

### 5. データ行（2行目以降）- 交互背景色
1. A2:W1000を選択
2. 「表示形式」→「交互の背景色」
3. スタイル: デフォルト（白/薄いグレー）

### 6. R〜W列 - テキストの折り返し
1. R2:W1000を選択
2. 「表示形式」→「テキストの折り返し」→「折り返す」

## 共有設定（重要）

スプレッドシート作成後、GASのサービスアカウントに**編集者権限**を付与してください。

1. 「共有」ボタンをクリック
2. サービスアカウントのメールアドレスを追加
   - 形式: `xxx@xxx.iam.gserviceaccount.com`
3. 権限: **編集者**

## スプレッドシートIDの確認方法

スプレッドシートのURLから取得:
```
https://docs.google.com/spreadsheets/d/【ここがスプレッドシートID】/edit
```

例:
```
https://docs.google.com/spreadsheets/d/1A2B3C4D5E6F7G8H9I0J/edit
→ スプレッドシートID: 1A2B3C4D5E6F7G8H9I0J
```

このIDをGASコードに設定します。
